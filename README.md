# ATAGUL IDE’da bilinmesi gereken temel kod bloklarını
### ata:
Bu komut sayıları veya değişkenleri belirtmek için kulanılır.
#### ÖRNEK:
##### ata x = 100

##### ata y = 50

### eger:
Bu komut bir koşul doğruysa bir işlemi çalıştırmak için kulanılır.
#### ÖRNEK:
ata x = 100

ata y = 50

ata çarpma = x*y

##### eger çarpma <100:

### yaz:
Bu komut ekrana yazı veya değer göstermek için kullanılır.
#### ÖRNEK:
ata x = 100

ata y = 50

ata çarpma = x*y

eger çarpma <100:

##### yaz "100'den küçük"

### bitir:
Bu komut programı tamamen durdurmak / sonlandırmak için kulanılır.
#### ÖRNEK:
ata x = 100

ata y = 50

ata çarpma = x*y

eger çarpma <100:

yaz "100'den küçük"

##### bitir
